    • Version control

    Git

    Git is a software used to keep track of the changes made in your files, so you have a record of your versions. Git allows you to have multiple people making changes in different parts of your project simultaneously.

  • Follow this guide to get started with git

    • GitHub

    GitHub is a hosting website where developers often store and manage their code. GitHub uses Git, so developers can also track and control changes to their project. GitHub offers a user friendly interface and makes it easy for anyone to get started.

  • Create a GitHub account

    Amazon Web Services

    AWS is one of many cloud computing service platforms. AWS securely offers services such as database storage and computing power. AWS is easy to use, scalable, and has a great reputation. Depending on your needs, AWS can be either very cheap or very expensive.

  • Get started with AWS

    • Microsoft Azure

    Azure is another cloud computing service platform. Azure and AWS practicly offers the same service to their customers. Which provider to pick depends on your needs, and in most cases you can be evenly satisfied with whichever you choose.
